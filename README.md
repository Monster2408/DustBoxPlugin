# DustBoxPlugin

## 概要
このPluginはMonsterLifeServerにあるDustBox機能だけを取り出したものです。

## 使い方？
Pluginをサーバーのプラグインフォルダにぶち込むだけ。

## 注意事項
コマンドは`/dustbox help`で確認可能です。
また、コンフィグ類の編集時、一部当Plugin独自の変数を用意していますのでお気を付けください。
※それを無視すると動作しなかったりします。
